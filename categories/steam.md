# STEAM – Integração com Ciência, Tecnologia, Engenharia, Artes e Matemática

Softwares que articulam música com outras áreas do conhecimento.

---

## Softwares disponíveis

### Sonic Pi

- **Descrição:** Ambiente de live coding para criação musical com Ruby.
- **Licença:** MIT
- **STEAM:** Tecnologia / Matemática / Artes
- **Sugestão pedagógica:** Algoritmos musicais, pensamento computacional, performance.
- **Download:** [sonic-pi.net](https://sonic-pi.net)

### Music Blocks

- **Descrição:** Programação musical visual baseada em blocos (Sugar Labs).
- **Licença:** GPLv3
- **STEAM:** Matemática / Tecnologia / Artes
- **Sugestão pedagógica:** Introdução à programação musical para iniciantes.
- **Download:** [musicblocks.sugarlabs.org](https://musicblocks.sugarlabs.org)

### Pure Data (Pd)

- **Descrição:** Linguagem de programação visual para som e multimídia.
- **Licença:** BSD
- **STEAM:** Engenharia / Ciência / Artes
- **Sugestão pedagógica:** Síntese sonora, criação de instrumentos digitais, interatividade.
- **Download:** [puredata.info](https://puredata.info)
