# Sonic Pi

- **Categoria:** STEAM / Live Coding
- **Licença:** MIT
- **Site oficial:** [sonic-pi.net](https://sonic-pi.net)

---

## Descrição

Sonic Pi é um ambiente de programação musical criado por Sam Aaron. Permite criar música ao vivo escrevendo código em Ruby, promovendo o pensamento computacional e a criatividade sonora.

---

## Conexões STEAM

| Pilar | Conexão |
|-------|---------|
| **Tecnologia** | Programação em Ruby, algoritmos, live coding |
| **Matemática** | Padrões, ritmos, proporções, loops |
| **Artes** | Composição, improvisação, performance |
| **Ciência** | Acústica, frequências, ondas sonoras |
| **Engenharia** | Lógica de sistemas, design de som |

---

## Sugestões pedagógicas

1. **Introdução:** Criar uma melodia simples com `play` e `sleep`.
2. **Intermediário:** Usar loops e condicionais para criar variações rítmicas.
3. **Avançado:** Integrar com sensores (Arduino) para controlar parâmetros em tempo real.
