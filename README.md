# REMUS Livre

**Repositório de Recursos Educacionais Abertos para Educação Musical**

Bem-vindo ao REMUS Livre! Este catálogo reúne softwares livres e educacionais para educação musical, organizados por categoria, com informações sobre licenciamento, aplicações STEAM e possibilidades pedagógicas.

---

## Categorias

- 🎼 **[Notação Musical](categories/notacao-musical.md)** – MuseScore, LilyPond, TuxGuitar
- 🎧 **[DAWs (Estações de Áudio Digitais)](categories/daws.md)** – LMMS, Ardour, Audacity
- 🤖 **[IA Musical](categories/ia-musical.md)** – Moises, AIVA, Magenta
- 🧠 **[STEAM](categories/steam.md)** – Criação, tecnologia e arte integradas
- 📚 **[Educação Musical Geral](categories/educacao-musical.md)** – GNU Solfege, GCompris, LenMus

---

## Como usar este catálogo

1. Navegue pelas categorias acima.
2. Cada software tem informações sobre:
   - Descrição e funcionalidades
   - Licenciamento (GPL, MIT, etc.)
   - Conexões com os pilares STEAM
   - Sugestões de uso em sala de aula
   - Link para download oficial
